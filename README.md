## About The Project

<br />

주요 기능:

- 로그인을 통해 개인 스케줄을 저장, 관리하며 언제든 확인하고 수정할 수 있습니다.
- 월, 주, 일 단위의 달력 형식으로 일정을 한눈에 확인하고 편리하게 관리하며 일정 태그를 활용하여 원하는 일정만 필터링할 수 있습니다.
- 일정의 반복 여부와 반복 주기를 설정하여 반복되는 일정을 효율적으로 관리할 수 있습니다.
